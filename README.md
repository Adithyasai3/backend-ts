# Backend API (Node.js + TypeScript)

A simple backend API built using Node.js, TypeScript, and Express to understand how servers, routes, and data handling work.

## Features
- GET /health
- POST /users
- GET /users
- In-memory data storage (no database yet)

## Tech Stack
- Node.js
- TypeScript
- Express

## How to Run Locally

```bash
npm install
npx ts-node-dev src/server.ts
